# Client development

### Building the client

- [Install Deno](https://deno.land/#installation)
- `cd Website`
- run `deno task build`

### For development

- [Install Deno](https://deno.land/#installation)
- `cd Website`
- run `deno task dev`
- Visit http://localhost:8080/src/ in your browser
- Make some changes and refresh the page
